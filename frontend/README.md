# 리트의신

AI 기반 리트 시험 학습 앱

## 프로젝트 소개

리트의신은 리트 수험생들을 위한 AI 기반 개인 맞춤형 학습 앱입니다.
리트 수험생들이 집리트를 통해 실제 합격선에 도달할 수 있도록 도와주는 것이 목표입니다.

## 주요 기능

- AI 기반 진단 평가
- 일일 모의고사
- 학습 대시보드
- 오답 노트
- 이용권 관리
