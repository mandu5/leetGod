🚀 실행 방법
프로젝트 설정:
chmod +x setup.sh
./setup.sh

백엔드 실행: ./run_backend.sh

cd backend && python -m uvicorn main:app --host 0.0.0.0 --port 8000 --reload

프론트엔드 실행: ./run_frontend.sh

cd frontend && flutter run -d chrome --web-port=3001 --hot
